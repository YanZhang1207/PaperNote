# 做什么
- 设计`适用隔离栅驱动器`的`自适应`且`补偿时钟抖动`的死区时间电路

# 为什么做
- 现有用于隔离栅驱动器的自适应死区时间电路需要额外pin端，设计复杂
- 现有用于隔离栅驱动器的自适应死区时间电路受时钟抖动影响严重，会导致shoot-through或过量损耗
  
# 怎么做
- 思路：先自适应产生死区，再基于抖动对死区微调。
- 电路实现（图5）：`自适应死区`电路与`抖动补偿`电路`串联`;

## 自适应死区电路
- 思路：对已有pin信号（共模信号）进行采样，基于已有pin信号（共模信号）与死区极性的一一映射关系实现对死区极性检测，再基于检测结果对死区进行调制。
- 电路实现（图6）：由`采样电路`，`极性检测电路`和`调制电路`串联组成。

### 采样电路
- 电路实现（图6）：由共模瞬态抑制电路(CMTS)和谷值检测电路(Valley detector)组成，通过不同共模信号情况对$V_{VD}$的影响，实现基于$V_{VD}$对共模信号进行采样。

### 极性检测电路
- 电路实现（图6）：由$OP_2$和$CMP_1$组成，通过检测$V_{VD}$最低值与参考值的大小，实现对共模信号情况进而对死区情况检测。

### 调制电路
- 电路实现（图6）：由加减计数器、自配置死区时间延迟和可控延迟单元组成，通过死区情况检测结果对计数器进行调整，从而调整可控延迟单元的延迟长度。

## 抖动补偿电路
- 思路：建立信号$V_{OSC}$`抖动量`与`中间量`的映射关系，再建立`中间量`与$V_{LS}$`抖动量`的映射关系，从而建立$V_{OSC}$`抖动量`的$V_{LS}$`抖动量`的映射关系，选择特定的映射关系实现`补偿`。
- 电路实现（图8）：$V_{OSC}$抖动量与中间量映射电路和中间量与$V_{LS}$`抖动量`映射电路串联。

### $V_{OSC}$抖动量与中间量映射电路
- 电路实现（图8左）：通过charge pump2和S1建立$V_{CTR}$电压变化量与$V_{OSC}$抖动时长的映射关系。

### 中间量与$V_{LS}$抖动量映射电路
- 电路实现（图8右）：通过2N2P组成的可调充放电电荷泵电路建立$V_{OSC}$与充放电电流的映射关系，进而建立与$V_{JC}$抖动量的映射关系，最终建立与$V_{LS}$抖动量的映射关系。

# 结果
- 输入电压200V，电感47$\mu H$，电容$470\mu F$且负载电流$I_{LOAD}=2A$时，峰值效率97%。

# 意义
- 提出了无需额外pin端更简洁且能够补偿时钟抖动的适用于隔离型栅驱动器的自适应死区时间电路。

# 总结
- 优点：
  - 利用了隔离驱动器中已有信号与死区的映射关系，降低了设计复杂度和额外pin端。
  - 通过将抖动量进行补偿，实现了适用隔离驱动器的自适应死区。
- 缺点：
  - 慢速反馈环调制，速度慢，不适用快速切载。
  - 在接近最优死区时间时，用来映射死区极性的共模信号的检测精度不高，导致对死区极性的检测出错，最终无法准确调制到最优死区。
